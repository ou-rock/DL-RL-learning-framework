## Tutorial Structure

Each tutorial follows the four-tier learning path:

1. **Tier 1: Conceptual** - Quiz-based understanding
2. **Tier 2: Implementation** - Hands-on coding challenges
3. **Tier 3: Visualization** - Interactive exploration
4. **Tier 4: Scale** - GPU validation (optional)

## How to Use

1. Start with Tier 1 to build conceptual foundation
2. Progress to Tier 2 when quiz score >= 80%
3. Use Tier 3 visualizations to deepen understanding
4. Attempt Tier 4 to validate at scale
